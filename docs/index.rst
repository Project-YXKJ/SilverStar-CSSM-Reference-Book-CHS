================================
SilverStar's CSSM 参考手册
================================

.. highlight:: bash

本手册是银星科技缝纫机电控系统的综合资源--适用于操作人员和专业人员。

另外手册内容不是针对某个具体型号的产品而写的，而是为了说明功能，所以这里你可能找不到如何进入设置界面
或者输出和输入端口的连接图。

内容有三部分组成：

COMMON
  阐述功能，适用于所有的产品。

PRODUCT
  针对某一个特定产品的资料。

CONTACT
  开发者。

本手册由 :ref:`银星开发者 <devguide>` 维护，所包含的信息和提示将确保长期可靠
的运行，如果您发现有不一致或需要改进之处，欢迎指正！
