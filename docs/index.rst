================================
SilverStar's CSSM 参考手册
================================

.. highlight:: bash

本书是银星科技缝纫机电控系统的综合资源--适用于新用户和有经验的用户。

不是针对某个具体型号的产品而写的，而是为了说明功能，所以这里你可能找不到如何进入设置界面或者
输出和输入端口的连接图。

内容有三部分组成：

COMMON
  阐述功能，适用于所有的产品。

PRODUCT
  针对某一个特定产品的资料。

CONTACT
  开发者。

本手册由 :ref:`银星开发者 <devguide>` 维护，如有错误，欢迎指正！
